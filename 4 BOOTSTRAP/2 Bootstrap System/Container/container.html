Container vs. Container Fluid


Container

The .container class provides a responsive fixed width container. This whole web site is made with container class. It organizes the whole web site together and puts everything into a like box that organizes them.


Container Fluid

The .container-fluid class provides a full width container, spanning the entire width of the viewport. It covers and spreads out trough the whole page making sure that everything is covered. You might want to use container fluid for like end page images, youtube videos, etc... There is an example under this text.
